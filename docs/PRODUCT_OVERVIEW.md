## ⚙️ How It Works

### 1️⃣ Data Ingestion & Validation

-   Accepts any CSV file\
-   Auto-detects headers and structure\
-   Displays dataset preview\
-   Validates format and size

### 2️⃣ Automated Data Cleaning

-   Removes empty columns\
-   Detects data types (numeric, categorical, datetime)\
-   Handles missing values\
-   Detects duplicates\
-   Logs every modification

### 3️⃣ Auto EDA Engine

Generates structured profiling including:

-   Row & column count\
-   Per-column statistics (min, max, mean, median, std)\
-   Missing value counts\
-   Unique value counts\
-   Most frequent categorical values\
-   IQR-based outlier detection\
-   Correlation highlights

### 4️⃣ AI-Selected Visualisations

Supported visualisations:

-   Line (time trends)\
-   Bar (category comparison)\
-   Histogram (distribution)\
-   Scatter (numeric relationships)\
-   Pie (when meaningful)

Each chart includes an explanation of why it was selected.

### 5️⃣ AI Executive Insight Engine

The LLM generates:

-   5 Executive Insights\
-   3 Business Risks\
-   3 Opportunities\
-   3 Actionable Recommendations

Tone: - Business-focused\
- Decision-ready\
- Clear and concise
